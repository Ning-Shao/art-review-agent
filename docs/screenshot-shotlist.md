- 截图名称：产品首页总览

## 拍摄建议

- 桌面截图建议使用同一浏览器、同一窗口宽度，优先 1440px 或 1280px 宽。
- 文件名应与 `product-guide.md` 中的 Markdown 图片路径一致，统一放入 `docs/assets/screenshots/`。
- 对“规划中”功能不要拍成已实现功能；如需表达未来能力，应使用架构图或明确标注“规划中”。
- 涉及 Mock AI 的截图建议保留“Mock / 当前版本 / 模拟回复”等上下文，避免被误解为真实 AI 输出。
- 局部放大图应保留足够周边上下文，让读者能看出它属于工作台的哪个区域。
